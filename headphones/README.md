# Headphones - ALX HTML/CSS Project

## Description
A responsive headphones company website built with HTML, CSS, and JavaScript.

## Project Structure
- 0-index.html: Header and Hero section
- 0-styles.css: Styles for header/hero
- 1-index.html: What we do section
- 1-styles.css: Styles for services
- 2-index.html: Our results section
- 2-styles.css: Styles with pentagon shapes
- 3-index.html: Contact us form
- 3-styles.css: Form styles
- 4-index.html: Complete website with footer
- 4-styles.css: Footer styles
- 6-index.html: CSS pentagons demonstration
- 6-styles.css: Pure CSS pentagon styles
- 7-index.html: Website with animations
- 7-styles.css: CSS animations
- 8-index.html: Responsive hamburger menu
- 8-styles.css: Mobile menu styles
- 8-script.js: JavaScript for mobile menu

## Images
- headphones_hero_1.jpg: Hero background
- headphones_hero_2.jpg: Results background

## Features
- Mobile-first responsive design
- Hamburger menu for mobile
- CSS animations and transitions
- Pure CSS pentagon shapes
- Form validation
- Cross-browser compatible
