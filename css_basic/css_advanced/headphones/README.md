# Headphones Webpage Implementation

## Project Overview
This project involves implementing a responsive webpage from scratch based on a Figma design, using pure HTML and CSS.

## Design Source
- Designer: Nicolas Philippot
- Figma File: Provided by ALX

## Requirements
- No external CSS frameworks
- No JavaScript (until Task 8)
- Responsive design (mobile breakpoint: 480px)
- Max content width: 1000px centered
- Hover effects: #FF6565 for links, opacity: 0.9 for buttons

## Technologies Used
- HTML5
- CSS3 (with CSS Custom Properties)
- Fonts: Source Sans Pro
- Font Icons: Holberton School Icon Font

## Project Structure
- Task 0: README and setup
- Task 1: Header/Hero Section
- Task 2: What We Do Section
- Task 3: Our Results Section
- Task 4: Contact Us Form
- Task 5: Footer
- Task 6: CSS Pentagon Shapes (No Images)
- Task 7: Animations
- Task 8: Hamburger Menu with JavaScript

## How to View
Open any of the HTML files in a modern web browser.

## Features
- Fully responsive design
- CSS animations and transitions
- Accessible navigation
- Pure CSS shapes
- Mobile hamburger menu
