# CSS Advanced Project

## Overview
This project creates a fully styled webpage using HTML and CSS based on a Figma design.

## Objectives
- Understand CSS basics and specificity
- Apply styles to HTML elements
- Build sections: header, banner, quotes, videos, membership, FAQ, footer

## How to Run
Open `index.html` in your browser. Ensure `styles.css` is linked.

## Files
- `index.html` → HTML structure
- `styles.css` → all styles
- `README.md` → project description
